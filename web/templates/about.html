Ziotiki Bot

Un bot de Discord multifuncional que combina moderación, entretenimiento e inteligencia artificial generativa. Diseñado para enriquecer la experiencia de cualquier servidor con comandos útiles y conversaciones naturales potenciadas por OpenAI.

🟢 Activo 🐍 Python 🧠 GPT-4.1-mini ⚡ discord.py
📋 Información general
  • 🤖 Nombre Ziotiki Bot
  • 🏷️ Prefijo de comandos =
  • 🧠 Modelo de IA GPT-4.1-mini
  • 🗄️ Base de datos PostgreSQL
  • 🌐 Framework web Flask + Gunicorn
  • 🐍 Lenguaje Python 3
🧩 Módulos (Cogs)
  • 🛡️ Admin Moderación
  • 🎉 Fun Entretenimiento
  • 📡 General Utilidades básicas
  • 🧠 IA Inteligencia Artificial
  • ⚠️ Command Error Manejo de errores
⚙️ Tecnologías utilizadas
🐍 Python 3 🤖 discord.py 🧠 OpenAI API 🗄️ PostgreSQL 🌐 Flask 🚀 Gunicorn / WSGI 🔐 python-dotenv 📦 psycopg2 🔁 asyncio 📝 logging
✨ Capacidades destacadas
🧠

IA con memoria

El comando =ask recuerda el historial de tu conversación para dar respuestas más contextuales.

🔧

Prompts por servidor

Cada servidor puede tener su propio prompt personalizado con =setprompt, dándole identidad única a la IA.

🛡️

Moderación completa

Silenciar, expulsar y banear miembros con validaciones de jerarquía de roles para evitar abusos.

📊

Dashboard web

Panel de control en tiempo real con estado del bot, registros de errores y actualizaciones de prompts.
